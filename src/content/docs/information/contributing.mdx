Want to contribute to the wiki? Here's how!

### Option 1

If you're familiar with [GitHub](https://github.com/LumaLibre/wiki/issues), open an issue on the
repository with your suggestions or changes. We'll review it and merge it if it fits the wiki!

### Option 2

If you're not familiar with GitHub, you can join our [Discord](https://lumamc.net/chat) server and
suggest your changes in the `# üìö‚îÉsuggestions` channel. We'll review it and make the changes for
you!

### Option 3

For technical users, you can fork the [repository](https://github.com/LumaLibre/wiki) and make your
changes. Once you're done, open a pull request and we'll review it!

<br />
